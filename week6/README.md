# 💧 Sepolia Faucet DApp Project

이 프로젝트는 Ethereum Sepolia 테스트넷을 기반으로 작동하는 스마트 컨트랙트 기반의 Faucet(수도꼭지) 서비스입니다.

## 🛠 기술 스택
- **Smart Contract**: Solidity
- **Frontend**: HTML5, Tailwind CSS
- **Library**: Ethers.js (v6)
- **Network**: Ethereum Sepolia Testnet

## 🚀 주요 기능
- **MetaMask 연결**: 사용자의 개별 지갑 연동
- **잔액 실시간 조회**: 스마트 컨트랙트에 충전된 Sepolia ETH 잔액 표시
- **출금 기능**: 버튼 클릭 시 컨트랙트에서 0.1 ETH를 사용자 지갑으로 전송
